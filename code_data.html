Code and data

An incomplete list of the things I have been working on. All code and data associated with my publications should be available with the paper, contact me if you are looking for something in particular.

  • nematode_global: data and code used for our study on global biogeographic patterns of soil nematodes. Full text here, GeoTifs here.
  • nematode_volume: ballparking the volume of the world’s soil nematodes. In total, there are 1337.603 Billion cans of worms on the planet.
  • color_palettes: selected color palettes that could be useful for geospatial mapping projects.
  • future_cities: a Shiny app showing the future climates of world’s major cities.
  • preprint_layout: to create basic documents in LaTeX.
